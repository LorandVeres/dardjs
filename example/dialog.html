Dardjs dialog example

Dialog box with css classes styled. You can have multiple on the screen on the same time, style each diferently

Enjoy

                    
funyFunction(){
    alert('Say Hi');
}                      
                        
dialog.init(
    $('#funybutton'),
    {
        css : ['container', 'head', 'close button', 'body', 'footer'],
        title : 'Dardjs dialog box',
        html : $('.dialog-box'),
        acton : {
            myevent : ['click'],
            triger : ['#id'],
            handler: [ funyFunction ]
        }
    }
);

This is a second istance of the dialog box. It is with an apart css style demontrating the posibility to make your life nicer

The body of this dialog box it is appended from one html object existing in the document, making an easy templating

In the first dialog box the body is provided by html string. Useful when the response come from server
